Notes

This version uses the official widget route so the site stays backend-free and deployable as plain static files.

  • Source for odds/spreads: The Odds API widget docs.
  • NBA sport key used by the widget: basketball_nba.
  • Markets shown: moneyline and spreads.
  • The site never sends your key to our own server. It is used only in the embedded widget URL from The Odds API.
  • If the board is empty, confirm your key, the sport is in season, and the selected bookmaker has listed games.
  • For a future custom backend, move the odds fetch behind a server and keep raw API keys server-side.
